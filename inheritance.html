Inheritance with Python
Write a program using Python that creates a class called "Structure". The "Structure" class should have attributes that store the type of structure and the square footage of the structure. Also, this class should have a method that prints out the information about the structure (Call this method "info"). Next, create a class called "House". The "House" class should inherit the Structure class and have 2 new attributes that store the number of bedrooms and the number of bathrooms. The "House" class should override the "info" method and print out all of the attributes that correspond to the "House" class. Lastly, create a class called Barn that inherits the Structure class and has a new attribute that stores the number of stables in the barn. The "Barn" class should override the "info" method and print out all the attributes that correspond to the “Barn” class.
Check out this YouTube video on inheritance in Python.
Run the Following lines of code to test your program:
  • >>> h = House(“Split Level”, 1200, 3, 2.5)
  • >>> b = Barn(“Farm Barn”, 2000, 8)
  • >>> p = Structure(“Pavilion”, 1500)
  • >>> structures = [h,b,p]
