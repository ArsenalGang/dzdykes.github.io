GCD and LCM
Write a program using Python 2 or Python 3 that creates two methods. One method should calculate the Greatest Common Divsor (GCD) of any two numbers. The other method should calculate the Lowest Common Multiple (LCM). Both of these methods should accept two integers as arguments and return an integer value.
Check out this article on computing the GCD and LCM.
Run the Following lines of code to test your program:
  • >>> GCD(45,99)
    9
  • >>> LCM(12,14)
    84
  • >>> GCD(81,270)
    27
  • >>> LCM(11,15)
    165
